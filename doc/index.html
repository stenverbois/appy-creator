Editor

The editor is used to create the visual part of your application. The editor consists of several parts. On your left you have a list of components. With these components you create the application that you want. When you click on a component, the component will be visible on the panel in the middle.

When a component is selected on the smartphone panel, you can move it around by just dragging it to where you want. You can also resize the component if you want to. You do this by putting your cursor in the right bottom corner of the component. Then you just drag to the size you want. ZET GIF HIER ERGENS

On the right side of the editor you have the properties section. Here you can edit the properties of the components individually. More info about the components and their properties can be found in the components section. In this section you can also delete a component if you want.

On the top bar of the editor you have the following tabs. For the moment we are in the Design tab. When you're done with your design you can choose to add some logic to your app by going to the Logic tab. More info about the logic tab can be found in the logic section. The info tab contains info about the application itself, like the name of your awesome app, your name, the width and height of the application preview etc...

You can also save your app by clicking on the save tab. Once saved you can reload the app with the load tab. If you're done with making your app, you can upload it to the public by clicking on upload. When you do this you'll get a QR code back. This QR code is important! Other users will use this to download your application.

List

A list component is a collection of different components. The components of a list are created from a component template. This template can be created by clicking on Edit items in the properties of the list. In this template you insert the components each list element should have.

Attribute Info
Visible If the component is visible or not

Logic

The logic view is the view where you create the logic of your application! This is an important process because it makes your app do the things you want! The logic view consists of two parts. The first part is on the left. Here you can find the components that are present in your app. You can also find functions and these functions are the most important things in your logic! When you click on one of the components or functions, then a node will appear on the right side. This right side is the place where you connect your logic!
